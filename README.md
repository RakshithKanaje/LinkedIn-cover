This project is a custom-designed LinkedIn banner built using HTML and CSS. It features an animated gradient background, modern typography, and a clean layout that highlights personal branding and core technologies.

## Features

- Animated background using CSS keyframes and gradients
- Font styling using the Google Font "Urbanist"
- Responsive layout with Flexbox
- Interactive hover effects on technology logos
- Includes logos for GitHub, AWS, Docker, Selenium, Python, and Java

## Files Included

- `index.html`: Contains the HTML and embedded CSS styles
- `.svg` files: Technology logos used in the banner
- `README.md`: Project documentation

## How to Use

1. Clone the repository or download the project folder
2. Open `index.html` in any modern web browser or use a Live Server extension in Visual Studio Code for real-time preview
